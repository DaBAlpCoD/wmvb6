# wmvb6 — Projet Visual Basic 6 (1999–2001)

Ce dépôt contient l’archive originale du projet wmvb6, développé autour de l’an 2000 en Visual Basic 6.  
Il s’agit de l’ancêtre direct de MathEclec.

Les fichiers sont importés tels quels, sans modification, afin de préserver l’état historique du projet.  
Le code n’est plus compilable sur les environnements modernes, mais reste lisible et documente l’évolution du logiciel.
